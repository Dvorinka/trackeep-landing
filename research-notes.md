# Antimetal.com Reference Notes

## Layout Structure
1. **Top Nav**: Minimal, with logo left, links center, CTA right
2. **Hero Section**: Dark navy/deep blue background with gradient, large headline "Handle everything that happens after you deploy", subtitle, search/ask input
3. **Ship More Section**: White/light background, "Ship more, break less" heading, feature cards with mock UI
4. **Fix & Prevent**: Two-column layout with feature descriptions and mock UI cards
5. **Stats/CTA Section**: Blue gradient to dark, "Engineers spend 40% of their time maintaining", yellow CTA button
6. **Knowledge Section**: "Scale your knowledge to infinity", search bar mockup, knowledge base features
7. **Integrations Section**: "Connect your stack" with integration icons
8. **Social Proof**: Stats like "5x Faster resolution" and "234K+ Engineering hours saved"
9. **Final CTA**: "Be ready for whatever you launch next" with Book a demo button
10. **Footer**: Dark background, three columns (Company, Legal, Connect), compliance badges, large logo

## Visual Style
- Dark hero with deep navy gradient (#0a0e27 to blue to white)
- Light middle sections (white/off-white)
- Blue gradient transition sections
- Dark footer
- Cards with subtle shadows and rounded corners
- Mock UI elements (terminal-like, browser windows)
- Yellow/green accent for CTAs in some areas
- Primary blue accent throughout

## Typography
- Large bold headlines (likely Inter or similar sans-serif)
- Clean body text
- Monospace for code-like elements

## Key Design Elements
- Gradient backgrounds (dark navy → blue → white)
- Floating card mockups
- Code/terminal-style UI elements
- Subtle animations
- Clean spacing
